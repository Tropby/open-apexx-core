Setup

You are about to install Open Apexx Version (open-apexx-core {APEXX_VERSION}). This installation will help you to setup the directory rights and database. You will create an administator user and choose some other settings.

This installation contains the "open-apex-core" modules.

  • main
    Needed for the core to run.
  • mediamanager
    Needed to access files on the webspace.
  • modulemanager
    Needed to install other modules into the system.
  • user
    Needed to get access rights for the administation center.
